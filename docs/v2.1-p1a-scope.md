## 4. Surface freeze guard

Items 5 + 6 + 8 touch FROZEN surfaces (post-P1 ack envelope schema,
SSE per-conn queue cap, cassette envelope recorder). Each must be
additive-or-versioned:
- **Item 5** bumps `sig_v: 1 → 2` so pre-P1a v1 rows still validate
  under the v1 schema. P2/P3 verifiers MUST accept both v1 + v2 sigs.
- **Item 6** widens the queue and logs `QueueFull` — additive
  (existing behavior was silent drop; logging is observable
  deterioration disclosure, not a behavior change).
- **Item 8** changes the cassette `idx` arithmetic from
  `len(payloads) + 50` to `len(payloads) + actual_lm_count`. This
  is a numbering fix, not an envelope schema change — cassette
  readers disambiguate by `kind`, not `idx`.
