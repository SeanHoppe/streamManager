**References:**
- `src/stream_manager/lifecycle_bridge.py` — `list_active_jobs` and the SQL query layer

**Do-not-touch guard:** Protected symbols — `LifecycleBridge` outer surface (`/api/lifecycle/jobs` payload shape, dashboard lifecycle pane contract). Fix is in the SQL query layer only. Do not change method signatures, return-tuple shapes, or envelope schema.

```
grep -nE 'list_active_jobs|/api/lifecycle/jobs|LifecycleBridge|_seen' src/stream_manager/lifecycle_bridge.py dashboard/server.py
```

**Scope:**
