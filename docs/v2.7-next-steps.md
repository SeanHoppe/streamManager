## Cycle-type fallback (operator override at P0 fire)

If operator overrides default-lean at P0 fire, the following
dispositions apply:

**Path-F default override → Path-C (consolidation):**

- Row 2 (`phase-1-timeout-tighten.md`) becomes **DEFERRED v2.8**.
- Row 3 (`phase-1-shadow-synthetic.md`) stays **DEFERRED v2.8**.
- Seed v2.6-G renames Seed v2.7-G; step (2)+(3) carry to v2.8 P0.
- Amendment F freeze-lift defers to v2.8 P0.
- Cycle becomes 2-numbered-phase (P0 + P2 ship-gate), 0 work phases.
- Lever ledger HOLD at 2 production / 0 soak.
- Operator records the override rationale in `docs/v2.7-task-plan.md`
  §"Operator decisions" #1 (typical: scope uncertainty, alternation
  hygiene, lever-budget bound).

**Path-C default override → Path-F (feature):**

- Row 2 fires (`phase-1-timeout-tighten.md` minted ahead-of-fire per
  PR #195 precedent).
- Amendment F freeze-lift co-minted at v2.7 P0 OR separate ADR-18 PR.
- Cycle becomes 3-numbered-phase (P0 + P1 + P2), 1 work phase.
- Lever ledger BUMP 2 → 3.

**Either default override → maximal feature (2 work phases):**

- Row 2 + Row 3 both fire.
- Seed v2.6-C deferral count breaks at 4 cycles (rare; operator-
  bound, requires v10.3 stochastic-propensities dependency status).
- Cycle becomes 4-numbered-phase (P0 + P1 + P1a + P2), 2 work
  phases.
- Lever ledger BUMP 2 → 3 (Seed v2.6-G step (2)).

This fallback exists because alternation hygiene is NOT law; ADR-18
Amendment A §"Default lean rationale" permits two-cycle-in-a-row of
the same type when the candidate-lever backlog has a ready entry.

When a v2.7 PR lands:

1. Find the seed row this PR addresses (use compare-back row marker).
2. Mark `[x]` and append `LANDED PR #N (<merge-SHA>)`.
3. If the PR addressed a seed that was NOT in the fire-order, append
   `OFF-SCOPE — operator-bound; recorded for narrative only`.
4. If the PR closed a 🔴 row → drop the 🔴 prefix in the v2.7 backlog
   carry-forward at v2.7 P2 close-out.
5. At v2.7 P2 ship-gate close memory, re-run row-by-row pass: every
   row must be ✅ LANDED, ⏸ DEFERRED v2.8 with rationale, or ❌
   DROPPED with rationale.
