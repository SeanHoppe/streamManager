The glance (return-to-desk scan)

Nine background sessions humming; you step away for coffee. WITHOUT this, the rail shows generic live pips and a stale client-derived count -- to know which session needs a HITL decision you must select each lane and wait for the panes to re-fetch decisions + agents + jobs + hitl (4 roundtrips x N sessions of poll-hell), then mentally reconstruct "quiet or stuck?".

WITH it ON, one green wall with a single red ACTION 2 lane jumps out. You click it -- the HitlDock is already the right scope. Zero reconstruction. Expand any lane (the existing chevron) to read the pre-computed digest detail.

Paired label + color badge (M4 -- color is never the only channel)

QUIET green -- hitl_pending_count == 0 and no variance alert
VARIANCE amber -- governance_variance_alert (WARN, badge-in-place)
ACTION 2 red -- hitl_pending_count > 0

The chip text and the lane aria-label both state the reason, so a screen-reader user gets the identical verdict. The chip sits on its own light bg, so the fg-on-bg contrast is self-contained against the dark lane and the always-present word means a contrast failure can never hide a signal.

Calm + safety contract

  • The digest badge NEVER pulses or auto-foregrounds -- a variance / quiet state is badge-in-place only. The lone permitted M2 pulse stays reserved for a true foreground escalation (governance_negative_regression / desktop_pause / static-rule).
  • Read-only GET -- no new bus envelope, no frozen surface touched. The endpoint applies _reject_sm_own(session_id) so the digest can never describe SM's own session (G2 polarity); the rail still self-excludes (footer readout).
  • Identity is rendered FROM DATA (project_slug) -- placeholder names only, no monitored-project vocabulary (M16).
  • Flag OFF => the whole SessionHealthBadge / digest-poll path renders nothing and registers no poller (no-op gate). Flip the switch above to see it collapse back to the plain lane.

Keyboard path (unchanged)

  • The lane is already role=button tabindex=0 with Enter / Space select.
  • The badge adds NO new tab stop -- it is aria-hidden; the lane aria-label carries it.
  • Tab to a lane, Enter / Space to select it, then the separate expand button (aria-expanded) reveals the digest detail. The 2px amber focus ring (M17) still applies to lane + expand.
