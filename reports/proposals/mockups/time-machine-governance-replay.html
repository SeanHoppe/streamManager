Time Machine

Counterfactual governance replay. Before you commit a threshold or HITL-mode change live, see exactly which of the last hour's REAL decisions it would have flipped -- read-only, nothing persisted, no live session touched.

  1. Flip Time Machine ON in Settings > BETA features (default OFF).
  2. Set the window (last 1h / 6h / 24h) and the config delta, pre-filled from your current settings.
  3. Nudge the confidence floor to the trial value.
  4. Click Replay window -- one POST returns the diff matrix + a +N/-N tally.
  5. Click any CHANGED row to expand the original vs counterfactual reason.
  6. Optional: Export diff (.md) for a team review note (client-side, no server write).
