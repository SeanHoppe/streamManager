Operator settings

Quick-filter presets BETA -- default OFF, toggled in Settings

One named click sets all four knobs below. Cause sits above effect -- clicking a card cascades DOWN into confidence / HITL mode / timeout / pause. Every knob stays fully mutable; a preset is a shortcut, never a constraint. Hand-tune any knob and the active mode flips to "custom".

Confidence floor 60%

Advisory threshold below which a decision is flagged low-confidence in place (badge only -- never auto-foreground).

HITL mode SYNC

SYNC holds the message until you decide; ASYNC lets you decide and annotate without holding. There is no off state (M5).

Sync timeout 60s

How long a SYNC HITL hold waits for an operator decision before it times out. Drives the pending-row countdown.

Pause detection

When on, a detected desktop-orchestration pause auto-foregrounds a frame (one of the three true escalation triggers).

Build gated UI is approvable; the BUILD is gated on an ADR-18 amendment. Custom presets persist client-side in localStorage (key sm.next.presets) for the minimal BETA. The cross-session upgrade path uses the EXISTING additive sessions.settings JSON blob -- no new table, no FROZEN schema edit. The feature also inherits the BETA foundation from task #5 (registry + BetaToggles + flag gate). No new bus envelope -> no cassette_record.py / soak_driver.py change required.

Paired label + colour (colour is never alone -- ADR-18 M4)

  • active card -- accent fill --accent + heavier weight. The NAME and the literal micro-spec (0.95 . SYNC . 120s) are ALWAYS text, so the live posture reads with all colour stripped.
  • custom -- muted slate. Shown when the operator hand-tunes away from any preset; the header pill reads "MODE: custom" (label text, never colour alone).

How the preset behaves

  • Click a card (or Arrow + Space/Enter) -> all four knobs below snap; the header pill updates. One click, zero reload (M1/M18).
  • Hand-move ANY knob -> the active card clears + the pill flips to "custom" (the config no longer equals a preset).
  • "Save as preset" -> name it, Enter -> it joins the rail after the divider, persisted in localStorage (sm.next.presets).
  • Custom cards carry an "x" delete; built-ins do not.
  • radiogroup keyboard a11y; the global 2px amber focus ring lands on every card.
