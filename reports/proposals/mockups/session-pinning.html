B Sub-Agents Observed agents -- monitor only, no inter-agent gating 2 action 1 pinned

Paired label + colour (colour is never alone -- ADR-18 M4)

  • PINNED -- slate-cyan --pin-fg. Anchored / structural. Deliberately NOT amber.
  • action -- FROZEN amber --badge-ar-fg. Escalation / action-required (M3). Pin must never read as this.
  • live -- active in window --live-fg. Auto activity pin, not a sticky pin.

How the pin behaves

  • Hover / focus a row -> quiet outline pin glyph appears.
  • Click (or Enter / Space on the focused button) -> row hops to the PINNED group at row 1; glyph fills + gains a "PINNED" chip.
  • Focus stays on the button after re-sort -- pin several without losing your place.
  • "developer" has the OLDEST last_seen yet stays row 1 -- it never recedes.
  • Persists per-session in localStorage across reload (mocked here).
